Exemple Moyenne

Objectifs

Calculer la moyenne des éléments d'une liste d'entiers.

Version 1

float moyenne(int[] maListe)
{
   int i = 0;
   int somme = 0;
   while(i < maListe.length){
      somme += maListe[i];
      i++;
   }
   return somme / maListe.length;
}

Version 2

On sépare le calcule de la somme des éléments et la moyenne. On obtient ainsi 2 méthodes utilisables individuellement.

float sum(int[] maListe)
{
   int i = 0;
   int somme = 0;
   while(i < maListe.length){
      somme += maListe[i];
      i++;
   }
   return somme;
}

float moyenne(int[] maListe)
{
   return sum(maListe) / maListe.length;
}
