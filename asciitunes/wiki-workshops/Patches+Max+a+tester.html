max v2;
#N vpatcher 319 131 783 497;
#P window setfont "Sans Serif" 9.;
#P newex 193 172 72 9109513 osc-route /trailx;
#P newex 156 111 64 9109513 udp-read 3000;
#P newex 156 142 84 9109513 OpenSoundControl;
#P newex 193 198 26 9109513 print;
#P connect 2 0 1 0;
#P connect 1 1 3 0;
#P connect 3 0 0 0;
#P pop;

Remarque : remplacer l'objet udp-read par l'objet udpreceive (avec argument 3000, si port 3000). osc-route qui s'écrit OSC-route, est un objet qui doit être installé, il interprète les messages arrivant dans Open Sound Control, et dispatch / route / achemine leurs composantes individuellement.
